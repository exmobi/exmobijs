transObj(jsonObj)
该方法可以将一个json对象转换成json字符串，一般用于一些ajax提交中需要提交json字符串的情况以及一些其他的json操作，如： $u.transObj({'name':'nandy','age':'10'})
JSON相关介绍，JS中：
var jsonObj = {'name':'nandy','age':'10'};//这是一个JSON对象
var jsonArr = [{'name':'nandy','age':'10'},{'name':'lily','age':'8'}];//这是一个JSON数组
var jsonStr1 = "{'name':'nandy','age':'10'}";//这是一个JSON字符串，因为用引号引起来了 var jsonStr2 = "[{'name':'nandy','age':'10'},{'name':'lily','age':'8'}]";//这也是一个JSON字符串
alert(jsonObj)和alert(jsonArr)返回的都是object，alert(jsonStr1)和alert(jsonStr2)可以看到JSON的内容。
但是可以通过JSON格式操作只有jsonObj和jsonArr，比如alert(jsonObj.name)是可以的，alert(jsonStr1.name)会报错。
